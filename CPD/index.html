09/06/2021

CSS - Positioning

What have I learnt?

Use of CSS for positioning (Relative, Absolute & fixed). Learned the different Layout Design Methodologies of Graceful Degradation and Progressive Enhancement with the latter being thought to be the preferred method as it ensured that the design for mobile can be scaled up to the PC version of the webpage. Flexbox basics command operations were covered in relation to legacy design methods to convey the step change that has been brought by the introduction of this postioning technology.

How did I use this in practice?

We were introduced to cssbattle.dev as a platform for the utilisation of CSS to change the appearance of web data on the page. An example was worked through to instruct on the usefulness of these examples.

What did I find hard?

I was unaware of the border radius commands in CSS which were needed for the example. I require further self learning in this area.

16/06/2021

Module 2 final project

What have I learnt?

Set up from scratch the file structure for a Band promotion webpage for a performance at the LTU arms. Created file for index.htm, styles.css and scripts.js.

How did I use this in practice?

Gained familiarity with the separate sections of a webpage. Completing the relevant data through the use of Boilerplate data and completion of the relevant data in the appropriate tagged sections for the header, body and footer. Use of relevant markup in headers and paragraph tags in the body section in main and aside. External links used from Youtube and Spotify to link to video and audio content. Use of CSS to maniplulate the placement of html elements via functions. Responsive design through use of media queries to adapt the placement of the data dependent on the screen size.

What did I find hard?

This was fast paced and introduced a lot of concepts in quick succession. Although I kept up it gave little tinme to absorb the information for future use.
