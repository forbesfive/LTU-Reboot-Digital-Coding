09/06/2021

CSS - Positioning

What have I learnt?

Use of CSS for positioning (Relative, Absolute & fixed). Learned the different Layout Design Methodologies of Graceful Degradation and Progressive Enhancement with the latter being thought to be the preferred method as it ensured that the design for mobile can be scaled up to the PC version of the webpage. Flexbox basics command operations were covered in relation to legacy design methods to convey the step change that has been brought by the introduction of this postioning technology.

How did I use this in practice?

We were introduced to cssbattle.dev as a platform for the utilisation of CSS to change the appearance of web data on the page. An example was worked through to instruct on the usefulness of these examples.

What did I find hard?

I was unaware of the border radius commands in CSS which were needed for the example. I require further self learning in this area.
